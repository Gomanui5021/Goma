重要 ffmpegと言うアプリが必須です

ダウンロードしたファイルに同梱をしていますが
動かなかったら下記のコマンドを入力してください

winget install --id=Gyan.FFmpeg -e

を管理者のPowerShell、ターミナル、コマンドプロンプトでインストールできます

文字数音声鳴らしツール

指定した音声を文章の数文鳴らせます

～使い方～

入力音声ファイルに鳴らしたい音声を入れます

保存先ファイルに保存先を設定します

文章入力に任意の文章を入れてください
(下に何文字か書いてあります)

1文字あたりのミリ秒は基本弄らなくてOKです
(数値を入れないとめちゃくちゃ高速になったりします)


！注意！

音声を文字数回分音を鳴らすだけなので連続して鳴らしたいなら
音声自体をいじらないと出来ないから気を付けてね
(0.2秒ほどの音声にしないとダメです 音声編集アプリで短くしてから使ってね)
